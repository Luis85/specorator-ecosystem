faq_list:
  - question: What is the Specorator ecosystem?
    answer: Specorator is a suite of interconnected open-source projects covering UI visualization, an execution runtime, an agent library, and a workflow definition system. This hub is the central navigation and status point for all of them.

  - question: What is the current status of the projects?
    answer: "Specorator (the UI layer) is actively in progress. The other three projects — specorator-runtime, agentonomous, and agentic-workflow — are in the planned stage. Track progress in the Roadmap."

  - question: Where can I follow development?
    answer: All repositories are under github.com/luis85. Watch or star individual repos to get notified of updates, or check the Roadmap page on this hub.

  - question: Will this hub show live data from GitHub?
    answer: Not in V1 — all data is static and managed through content files. V2 will integrate the GitHub API to pull live commit dates, issue counts, and CI status automatically.

  - question: How is the hub deployed?
    answer: The hub is built with Astro and deployed to GitHub Pages via GitHub Actions. Every push to the develop branch triggers a build and deploy.
