faq_list:
  - question: What is the Specorator ecosystem?
    answer: Specorator is a suite of interconnected open-source projects that take any idea to shipped software using a 12-stage Agentic Development Lifecycle. It covers UI visualization (Obsidian plugin), an execution runtime, an agent library, and a workflow definition system. This hub is the central navigation and status point for all of them.

  - question: Who is this for?
    answer: Anyone who wants to build software with AI doing the heavy lifting — product managers, designers, founders, and developers. The recommended starting point (the specorator Obsidian plugin) requires no coding. Individual components are available for developers who want to build their own tools on top.

  - question: What is the current status of the projects?
    answer: "specorator (the UI layer) and agentic-workflow (the methodology) are both actively in progress. specorator-runtime and agentonomous are planned. Track live status and open issues on the Projects section of this hub — data is fetched from GitHub at build time."

  - question: Where can I follow development?
    answer: All repositories are under github.com/luis85. Watch or star individual repos to get notified of updates, or check the Roadmap page on this hub for milestone-level progress.

  - question: How is the hub deployed?
    answer: The hub is built with Astro and deployed to GitHub Pages via GitHub Actions. Pushes to the main branch trigger a verify run followed by deployment. The develop branch runs CI verification only.
